About Me

개발 철학 및 강점

저는 사용자 중심의 문제를 기술로 해결하는 것에 매력을 느끼는 프론트엔드 개발자입니다.

이번 **'스케쥴 메이트'** 프로젝트를 통해 특히 보여주고자 한 역량은, 대학생들의 시간 관리 스트레스 해소와 학습 효율 향상이라는 문제를 해결하기 위해 AI 기술을 직접 서비스에 녹여내는 것입니다.

React와 Flask를 활용한 풀스택 개발을 경험하며, 사용자 경험(UI/UX)과 데이터 기반 서비스 로직을 연결하는 역량을 성공적으로 보여주었습니다. 특히, Google Calendar API 및 GPT 기반 AI 알고이즘을 연동하여 실제 사용자의 일정 데이터를 활용한 스마트 자동 일정 배치 시스템을 구현하는 데 중점을 두었습니다.

Project: 스케쥴 메이트

복잡한 대학생 일정을 AI가 분석하여 최적의 학습 루틴과 시간 관리를 자동 설계해주는 스마트 스케줄링 서비스입니다. 이 섹션의 탭을 클릭하여 프로젝트의 상세 내용을 확인해보세요.

문제 정의

대학생들은 전공, 팀 프로젝트, 시험 등 복잡한 일정으로 인해 시간 관리 스트레스를 겪고 있습니다. 기존 캘린더 앱은 단순히 일정을 나열할 뿐, 우선순위나 학습 효율성을 고려하지 않아 비효율적입니다.

목표

사용자의 상황에 맞춰 AI가 자동으로 일정의 우선순위와 학습 루틴을 설계해주는 솔루션을 제공하여 시간 관리의 비효율성을 해소하고 학습 효율을 향상합니다.

아키텍처 및 설계

  • AI 모델: GPT 기반 AI 모델을 이용하여 일정의 중요도, 마감일, 선호도를 분석 후 자동 배치 로직 구현.
  • 데이터 연동: Google Calendar API를 통해 학사 일정 자동 연동.
  • Frontend: React(웹) 및 Flutter(모바일) 기반의 직관적인 사용자 인터페이스(UI/UX) 설계 및 개발.
  • Backend: Flask + FastAPI를 사용하여 AI 연동 및 데이터 분석 처리.
  • 데이터 관리: 사용자 일정 및 학습 패턴 데이터를 중앙 DB에 저장.

핵심 기능 1: AI 기반 스마트 우선순위 알고리즘

GPT 기반의 AI가 사용자의 일정 데이터를 분석하여 중요도, 마감일, 개인 선호도를 반영한 자동 일정 배치를 수행합니다. 이를 통해 학습 시간과 휴식 시간이 최적의 밸런스를 이루도록 계획합니다.

핵심 기능 2: Google Calendar API를 통한 학사 일정 자동 연동

Google Calendar API를 활용하여 학사 일정을 자동으로 가져오고, 사용자의 일정과 통합 관리할 수 있게 구현했습니다. 이를 통해 수동 입력의 번거로움을 줄이고, 실제 학사 일정에 맞춘 맞춤형 학습 플랜이 가능하도록 했습니다.
